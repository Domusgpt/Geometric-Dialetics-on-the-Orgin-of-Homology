# Citation Formatter

Format references for academic papers on Polytopal Projection Processing.

## Supported Formats

### APA 7th Edition (Psychology, Cognitive Science)
```
Author, A. A., & Author, B. B. (Year). Title of article. Journal Name, Volume(Issue), pages. https://doi.org/xxx
```

### IEEE (Robotics, Engineering)
```
[1] A. A. Author and B. B. Author, "Title of article," Journal Name, vol. X, no. Y, pp. XX-YY, Month Year.
```

### ACM (Computer Science)
```
Author, A. A. and Author, B. B. Year. Title of article. Journal Name Volume, Issue (Month Year), pages.
```

### Nature/Science Style
```
1. Author, A. A. & Author, B. B. Title of article. Journal Name Volume, pages (Year).
```

## Task

When given a reference or citation need:

1. **Look up** the correct bibliographic information
2. **Format** in the requested style (default: APA)
3. **Provide** in-text citation format
4. **Note** DOI or URL if available
